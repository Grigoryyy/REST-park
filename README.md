# REST-park
Тестовое задание для компании HFLabs

REST web-сервис для парковки машин, написан с помощью внутренних библиотек и модуля flask без стилей

Перед запуском основной программы (*app.py*) необходимо запустить *dependency_check.py*\
чтобы проверить наличие необходимых модулей. Файлы рекомендуется держать в одной папке,\
index.html должен находиться в папке templates.\
Примерная структура:\
- Папка_проекта:
  * app.py
  * backend.py
  * dependency_check.py
  * templates
    * index.html

Проект разбит на части, backend.py отвечает за основной функционал сервиса\
в нем расположены все функции для занятия парковочных мест и подсчета стоимости.\
app.py - основной веб-сервис на flask. Связывает воедино backend.py и index.html.\
index.html - отвечает за показ проекта в браузере.

Интерфейс программы прост.

1 версия - 1 поле предлагает ввести номер машины, в случае успеха появится характерное сообщение\
и оставшееся количество мест, в противном случае(если все места будут заняты) появится сообщение о неудаче.

2 версия - добавлено поле ввода номера машины для выезда с парковки, поле с указанием часа заезда\
и поле с указанием количества часов, проведенных на парковке. Также, если введен неверный номер машины,
то появится характерное сообщение об ошибке.

3 версия - снижена цена в определенные часы парковки, а именно с 10 вечера до 7 утра.

4 версия - добавлена возможность для парковки грузовых машин. Для определения машины добавлен 
radiobox, при этом легковая машина, в случае отсутствия мест на парковке для легковых автомобилей\
может заехать на парковку для грузовых (если там осталось место), грузовые могут парковаться только\
на парковке для грузовых автомобилей.
